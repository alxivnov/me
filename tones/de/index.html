How to set a song as an iPhone tone?

Download on the App Store



1. Create a copy of a song with duration 40 seconds or less.

  1. a. Find the song in the iTunes and select "Informationen" in its context menu.
  2. b. On the "Optionen" tab set the start and the stop time and click OK.
  3. c. In the iTunes menu select sequentially items "Ablage", "Konvertieren" and "AAC-Version erstellen".
  4. d. A copy of the song will appear, select "Im Finder anzeigen" in its context menu.

2. Change the file type to "mp4" and import it to iPhone.

  1. a. In the context menu of the copy select item "Informationen".
  2. b. Change file extension to "m4r" in the appeared window and close it.
  3. c. Connect iPhone to Mac and select it and then Töne in the left menu.
  4. d. Drag the ringtone to the iTunes.

3. Set the new ringtone on your iPhone.

  1. a. Open settings on the iPhone or the iPad.
  2. b. Sequentially select items "Töne" and "Klingelton".
  3. c. Then find and select previously created ringtone.
  4. d. Check out a "Store" button in the top right corner and make conclusions.
